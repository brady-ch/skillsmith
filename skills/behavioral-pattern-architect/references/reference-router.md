## Route

- Need a quick pattern pick or need to distinguish similar patterns: load `pattern-selection.md`
- Need family-level framing before picking a pattern: load `behavioral-pattern-overview.md`
- Need request routing among handlers: load `chain-of-responsibility.md`
- Need packaged actions, undo, or deferred work: load `command.md` or `memento.md`
- Need direct notification or brokered fan-out: load `observer.md` or `publish-subscribe.md`
- Need runtime policy, mode, or workflow variation: load `strategy.md`, `state.md`, or `template-method.md`
- Need grammar evaluation or structure traversal: load `interpreter.md`, `iterator.md`, or `visitor.md`
- Need centralized interaction or rule/specialist composition: load `mediator.md`, `specification.md`, `null-object.md`, or `blackboard.md`
