# Skillsmith (nano)

省言為本。載入務少。

**Invariant**

- Catalogue 於此 repo：`SKILLSMITH_REPO_ROOT` → `catalog/catalog.toml`。
- 路由：**MCP** `skillsmith mcp serve`（`skillsmith_route_trace` / `skillsmith_recommend`）或 CLI：`skillsmith recommend --intent "<task>" --format json --limit 4`。
- 各 skill：`SKILL.md` → `references/reference-router.md` → **一** `.md`；勿載全 `references/`。
- 行文縮 `compression-skill-designer`；不可逆、安危、不清晰時詳。
- 規約：`docs/token-first-spec.md`。

**Full bootstrap**：`SKILLSMITH_HOOK_BOOTSTRAP=full` → 載入整份 `skills/using-skillsmith/SKILL.md`。（人讀：`references/best-practices-english.md` / `references/best-practices-wenyan.md`。）
