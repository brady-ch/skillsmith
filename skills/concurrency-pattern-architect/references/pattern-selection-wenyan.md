# 並術 快路由

## 圖
- 步寄主動對：Proactor
- 趣就緒反應：Reactor
- 互斥臨界：Lock / Monitor
- 讀寫割：Readers-Writer lock
- 屏障同步：Barrier
- 雙查建：Double-checked locking
- 觀而待：Guarded suspension / Balking
- 旗限流：Semaphore
- 工複用：Thread pool
- 域存線：Thread-local storage
- 表單步：Active object
- 合流：Join pattern
- 定時務：Scheduled task / Scheduler

## 淆
- Proactor 完後告；Reactor 緒告行
- Balking 不願等；Guarded 願等

## 訣
- 明模（反應、主動、排）乃擇
- 能消息則減共變
