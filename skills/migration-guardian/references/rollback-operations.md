# Rollback Operations (TOON)

## T - Trigger

Use when rollback readiness and operational safety are primary concerns.

## O - Objective

Specify how to detect migration failure early and revert safely.

## O - Output

Include:
1. rollback trigger thresholds
2. rollback command sequence
3. data integrity verification after rollback
4. stakeholder communication checkpoints

## N - Navigation

Prioritize:
1. reversibility of each migration step
2. immutable audit trail of migration actions
3. objective health gates before proceeding
4. rollback drills before production rollout
